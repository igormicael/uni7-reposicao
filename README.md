# uni7-reposicao

Projeto que visa repor estoque da aplicação [Loja Virtual](https://github.com/igormicael/uni7-loja-virtual)

### Prerequisitos

- Java 11
- Maven
- Wildfly 16


### Instalação Docker

- Gerar a imagem do projeto Reposicao, através dos comandos:
```sh
docker build -t reposicao .
```
